PeerProgramming

We want to help JavaScript newcomers build a bridge, from being interested to contributing to the community, by introducing them to more experienced peers in a relaxed environment.

If you feel you are committed to putting in the effort to learn JavaScript, we, in turn, are committed to helping you find the tools to do it â€“ in good company.

JavaScript has been said to be the lingua franca (working language) of the Web. With the rising popularity of Node.js for server-side programming, MongoDB for document storage and JSON for data interchange, JavaScript knowledge can be used in much more than just scripting Web sites.

The idea of HelsinkiJS PeerProgramming is to introduce dedicated JavaScript newcomers with more advanced peers who still have their first steps relatively fresh in their memory, and seasoned JavaScript professionals from whom to ask for additional information. This is a great change for the more experienced peer to learn by teaching.

The premise for this to work is that all attendees have a strong commitment to doing the individual work needed for real learning to occur. This is why attendees need to first complete the interactive Getting Started with Programming course (free) on CodeAcademy first, which represents the "preparation at home" part of learning by teaching.

If you are willing to take on the challenge of learning JavaScript, you now have a clear path for it.

  1. Follow @helsinkijs
  2. Tweet @helsinkijs with hashtag #peerprogramming that you want to join (tweet button below)
  3. We'll contact you with detailed instructions how to apply
  4. Receive a confirmation that your application has been approved
  5. Show up at the event with your laptop
  6. Welcome!
